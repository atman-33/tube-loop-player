# Testing Policy

## Test Requirements
- Write tests for business logic and data processing functions
- Test files: `*.test.ts` or `*.integration.test.ts`
- Run tests: `npm test`

## Coverage Focus
- Data synchronization logic (playlist, pinned songs)
- Conflict resolution and data comparison
- Critical user flows

## Testing Tools
- Vitest (test runner)
- @testing-library/react (component testing)
